# Azure Polybase

- ELT tool for bulk data.

- Removes complexity for big-data ingestions and processing by offloading calculations to the cloud.

- Use Polybase to:
    - Apply stored procedures.
    - Apply labels.
    - Apply views.
    - Apply SQL queries.
    - Ingest and process data.
